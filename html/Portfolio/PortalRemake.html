Portal Remake
University Project
For this project we where provided with an engine that the teachers wrote. And for every new topic, we got new files to add and add some extra code to make the new implementation work.
On top of all this we had to make a game. I chose Portal, since I heard about the game but never saw what is was all about, and when I saw it, I knew I had to try and make it. Although it is not completely like it should be.
Things we had to learn.
Skinning, Materials, Shadowmapping, Animation, Nvidia Physx
