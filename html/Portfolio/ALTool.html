Q*Bert Remake
University Project
The main purpose of this project was to create our own engine, we did get some start files, and show that our engine works by making a game with it that the teachers chose. We had to implement certain topics to do this. Most of the topics where chosen from Robert Nystrom's book Game Programming Patterns(*), with the most important topic, Components over Inheritance.
A problem I faced, was the grid. It has no square cells, but hexagon cells. Thanks to a site(*), I could quickly determine how I had to tackle this problem. I chose the "double-width technique", since it would be easier for the movement of entities if the next cell parameter always has the same value for the axis, but only a different sign depending on the direction.
